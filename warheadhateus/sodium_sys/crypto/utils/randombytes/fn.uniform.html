pub fn uniform(upper_bound: uint32_t) -> uint32_t

The uniform() function returns an unpredictable value between 0 and upper_bound (excluded). Unlike random() % upper_bound, it does its best to guarantee a uniform distribution of the possible output values.

The uniform() function safely wraps the randombytes_uniform() function.

Examples

use sodium_sys::crypto::utils::randombytes;

let r0 = randombytes::uniform(10);
assert!(r0 < 10);
